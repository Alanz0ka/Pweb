# Pweb

Um projeto web simples desenvolvido com HTML, CSS e JavaScript.

## 📋 Descrição

Este projeto é um site estático construído com tecnologias web fundamentais, sem dependências externas ou frameworks complexos.

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura e semântica
- **CSS3** - Estilização e layout
- **JavaScript (ES6+)** - Interatividade e lógica

## 📁 Estrutura do Projeto

```
Pweb/
├── index.html
├── css/
│   └── styles.css
├── js/
│   └── script.js
└── README.md
```

## 🚀 Como Usar

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` em seu navegador
3. Não há dependências a instalar!

## 📝 Arquivos Principais

- `index.html` - Página principal do site
- `css/styles.css` - Estilos e layout
- `js/script.js` - Scripts e interatividade

## 💡 Características

- Design responsivo
- Interface limpa e intuitiva
- Código bem organizado e comentado
